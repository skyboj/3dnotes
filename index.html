MemoryPalace

A digital memory palace for visual knowledge storage and spatial note organization

🚨 Problem

Most note-taking tools (like Notion or Obsidian) use linear or hierarchical structures. Users get lost in folders and links, lose context, and struggle with long-term memory and discovery.

💡 Solution

MemoryPalace is a spatial note-taking app that lets users build rooms and pin notes to actual walls inside a pseudo-3D environment. It's inspired by the method of loci — helping users remember, explore, and intuitively find knowledge by walking through their own "mental palace".

🛠️ MVP Features

  • Simple Markdown text notes
  • Navigation through rooms like classic 3D games
  • Click on a wall to pin and open a note
  • Persistent thumbnails attached to exact wall locations
  • Offline-first, stored locally in the browser (IndexedDB)

🎯 Target Audience

Students, researchers, writers, visual thinkers, neurodivergent users, personal knowledge management fans (PKM).

🆚 Competitors

Notion, Obsidian, Heptabase, memoryOS.
Key Advantage: spatial memory — users remember where ideas live, not just how they’re tagged.

🏗️ Project Status

Building MVP:
• Visuals via three.js
• UI in React + Markdown Editor
• Local-first storage (IndexedDB)
• Minimalist, distraction-free design
